# 前额叶训练中心 · Prefrontal Trainer

一个纯前端、零依赖的**前额叶认知训练套件**，系统覆盖注意、工作记忆、抑制控制、认知灵活性与规划等核心执行功能。当前内置四个经文献背书的训练范式，并采用**数据驱动的游戏库架构**，可低成本持续扩展更多范式。

> 本仓库是统一主仓库。早期独立版舒尔特方格已归档于 [`vestibular-training-game`](https://github.com/zengzhechun/vestibular-training-game)，其能力已完整包含于此。

## ✨ 已实现范式

| 范式 | 功能域 | 训练目标 | 关键特性 |
|------|--------|----------|----------|
| **舒尔特方格** | 注意 · 视觉搜索 | 视觉搜索速度、选择性注意 | 7 种图形布局（方形/圆形/三角形/多层嵌套/蜂巢/圆盘/变式）、正序/倒序、6 种可叠加干扰（颜色/字体/抖动/镜像/色块分区/旋转）、数字偏移、定时/计时双模式 |
| **Dual N-Back** | 记忆 · 工作记忆 | 工作记忆维持与在线更新 | 3×3 视觉网格 + 字母语音双通道、可调 N 等级/通道/试次/间隔、自适应升降难度、位置/声音匹配 |
| **Stroop 色词测验** | 抑制 · 抑制控制 | 抑制优势反应（自动阅读） | 字义与墨色冲突、4 色响应（红/绿/蓝/黄，支持 1–4 键）、一致性（混合/仅不一致/仅一致）、每题时限、正确率 + 平均反应双指标 |
| **Go/No-Go** | 抑制 · 反应抑制 | 抑制优势运动反应（忍住不按） | 绿「GO」反应 / 红「停」抑制、可调试次数量 / 反应窗口（0.5–1.4s）/ Go 占比（60–80%）、命中·正确抑制·误按·漏按四分类计分 + 平均反应 |

## 🚀 快速开始

无需安装、无需构建，双击 `index.html` 即可运行：

1. **直接双击** `index.html` 在浏览器中打开。
2. **或本地静态服务**（可选）：
   ```bash
   python3 -m http.server 8000
   # 浏览器访问 http://localhost:8000
   ```

## 🎮 玩法

- 顶部「训练游戏」卡片库按功能域（注意 / 记忆 / 抑制）分组，支持搜索；点击卡片切换范式。
- 在左侧设置面板选择该范式的参数（布局、模式、干扰、N 等级等），点击「开始」。
- 成绩自动保存到本机浏览器（localStorage），右侧历史面板提供统计、按游戏/布局/排序筛选的 SVG 趋势图（标注个人最佳记录线）。
- 快捷键：舒尔特 `空格`（开始·结束本轮·恢复）、`R`（重置）、`E`（结束本轮）；N-Back `F`（位置匹配）、`J`（声音匹配）、`空格`（开始·结束·恢复）、`R`（重置）；Stroop `1–4`（对应颜色）、`空格`（开始·结束·恢复）、`R`（重置）；Go/No-Go `空格`（反应·开始·恢复）、`R`（重置）。

## 🧩 架构（Phase 0：游戏库注册表）

新增范式只需在 `app.js` 的 `GAMES` 注册表中登记一项，声明其 `id / name / domain / 图标 / 描述 / 设置面板与舞台元素 id / 生命周期钩子（reset、idle）`，并在历史筛选 `filterGame` 中加入对应选项即可，**无需改动切换逻辑**。顶部卡片库（按功能域分组 + 搜索）由 `renderGameLibrary()` 自动生成。

```
prefrontal-trainer/
├── index.html   # 页面结构（游戏库 / 设置 / 舞台 / 历史）
├── style.css    # 响应式深色主题、玻璃拟态、动效
└── app.js       # 公共层（存储/音效/趋势图）+ 各游戏模块 + GAMES 注册表
```

## 🛠 技术说明

- **纯前端**：HTML + CSS + 原生 JavaScript，单页应用，无第三方库、框架或网络请求。
- **离线可用**：所有资源（含音效与语音合成）均在本地生成，双击 `file://` 即可运行。
- **数据安全**：成绩仅保存在本地浏览器，不上传任何服务器。
- **响应式**：自适应桌面与移动端，深色/浅色主题一键切换。
