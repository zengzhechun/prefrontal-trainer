# 前额叶训练热门小游戏列表

> 本列表由 DuMate 于 2026-07-10 检索整理，来源为网络公开信息，涵盖当前比较流行的前额叶（Prefrontal Cortex）训练小游戏及综合训练 App。

## 一、按训练能力分类的小游戏

| 训练能力 | 游戏/任务名称 | 玩法说明 | 代表产品/平台 |
|---------|-------------|---------|-------------|
| 工作记忆 | N-Back | 判断当前信息是否与前面第 N 轮出现的信息相同，持续更新工作记忆 | 《训练前额叶》、Brain Daily、Lumosity |
| 注意力集中 | 舒尔特方格 | 在打乱排列的数字方格中按顺序快速找到目标数字 | 《舒尔特专注力》、《训练前额叶》 |
| 抑制控制 | 色卡冲突（Stroop 测试） | 文字写"红色"但实际颜色为蓝色时，抑制直觉反应按规则选正确答案 | 《训练前额叶》、Brain Daily |
| 短时记忆 | 序列记忆 | 短暂展示一组数字/颜色/图形序列，记住后按序复现 | 《训练前额叶》、《脑力训练》 |
| 反应抑制 | Go/No-Go | 看到特定信号快速反应，其他信号则需抑制不动 | Brain Daily |
| 选择性注意 | Flanker 任务 | 聚焦中心箭头方向，忽略周围干扰箭头 | Brain Daily |
| 注意力切换 | 任务切换/卡片分类 | 根据规则变化在不同分类标准间快速切换 | Brain Daily、Lumosity |
| 逻辑推理 | 2048/华容道 | 数字合并路径预判、空间位移推理与步骤优化 | 《脑力达人》 |
| 全脑协同 | 密码破译/序列补全/三维旋转 | 左脑数理分析 + 右脑图像识别协同训练 | 《谁是最强大脑》 |
| 限时解谜 | 多机制闯关 | 60 秒内完成轨迹绘制、图形拼合、符号匹配等 | 《一分钟大脑挑战》 |
| 心流专注 | 呼吸节奏引导 | 动态视觉锚点 + 渐进式干扰抑制，3-5 分钟/局 | 《专注禅院》 |
| 多线程执行 | 模拟经营 + 时间分块 | 限定时间内完成植物培育、动物喂养等多任务操作 | 《专注时光》 |

## 三、非屏幕类补充训练方式

除电子游戏外，以下方式同样被研究证实可有效激活前额叶：

- **反向跳绳**：倒着跳、单脚跳、交叉跳等花式跳绳，多样化运动增加前额叶血氧供应，激活运动规划区
- **围棋 / 桥牌**：需要策略思考和快速决策，直接锻炼执行功能
- **双任务训练**：如一边散步一边心算，强制激活前额叶皮层
- **正念冥想**：每日 20 分钟专注呼吸，增强前扣带回功能
- **项目管理式训练**：让孩子负责一个完整项目（如策划家庭出游），锻炼提前规划、权衡利弊、考虑后果的能力
